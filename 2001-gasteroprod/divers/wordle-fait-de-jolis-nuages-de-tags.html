Pas mal de blogs ont dj parl ces derniers jours de Wordle, un service permettant de raliser via une applet Java1 des nuages de tags assez jolis.

Voyez par exemple celui produit par extraction automatique de mes tags del.icio.us :

Et cet autre produit avec les tags de Gastero Prod2 :

Mais ce que je n’ai vu personne signaler, c’est qu’ part produire une image, permettant certes de raliser de beaux t-shirts ou autres gadgets, ces nuages de tags ne peuvent malheureusement pas tre utiliss sur un site pour la navigation, puisque le rsultat n’est pas du tout cliquable.

Ce qui n’est bien entendu pas le cas du nuage de tags de Gastero Prod, qui n’est pourtant pas si moche non plus, par rapport ce qu’on peut voir ailleurs :

Mise jour du 1er juillet 2008 : Encore plus de nuages possibles :

  • Wordle permet maintenant d’indiquer une URL de blog ou de flux RSS, mais il semble qu’il utilise le contenu plutt que les tags, que je mets pour ma part dans les balises <category>
  • Il est aussi possible d’obtenir le texte correspondant ses tags Flickr pour le copier dans Wordle
